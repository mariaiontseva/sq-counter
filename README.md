# SmarterQueue Counter

A beautiful, animated counter displaying "13,000+ brands have chosen SmarterQueue"

## Features

- Smooth easing animation (easeOutExpo)
- Gradient background and text effects
- Sparkle effects during counting
- Responsive design
- Intersection Observer for scroll-triggered animation
- Number formatting with commas

## Usage

Simply open `index.html` in your browser to see the counter in action.

## Demo

The counter animates from 0 to 13,000+ with a smooth easing function and visual effects.
